# Support

Use GitHub issues for bug reports, feature requests, and documentation gaps.

Before opening an issue:

- search existing issues for similar reports.
- include the Memoria version and Node.js version.
- include the command you ran and the relevant output.
- describe whether the project is new or existing.

For workflow questions, include the goal you are trying to support with
Memoria, such as lower-token recall, durable project memory, or AI agent
slash-command usage.
